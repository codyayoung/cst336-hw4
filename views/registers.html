CPU Registers

Modern x86 processors have eight 32-bit general purpose registers, as shown in the figure.
Each register can hold 32 bits of data. Register names have lost their original historical purpose
and are a carryover from legacy computing days. For example, the ECX register was known as the "counter"
register because it was used to hold loop indexes.
Today, the type of data stored in each register is at the programmer's discretion.


Usage

The EAX, EBX, ECX, and EDX registers can be used as subsections.
Each of these registers can be divided into three sections as follows, using EAX as an example:

  1. EAX - 32 bits
  2. AX - 16 bits
  3. AH - 8 bits
  4. AL - 8 bits

When using subsections of these registers, they are called by different names. The least significant 2 bytes
of the EAX register are considered as the 16-bit register AX.
ESP and EBP are the only two registers still reserved for special purposes.
ESP is used to store the stack pointer in memory, while EBP stores the base pointer which
is a key part of calling subroutines in assembly.
