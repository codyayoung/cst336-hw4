What is Assembly?

An assembly language is a low-level programming language closely related to
a computer architecture's machine code instructions.

Assembly allows programmers to execute CPU instructions and maniuplate CPU memory, known
as registers, directly. Reading and writing assembly code is a valuable skill to have - it
enables programmers to optimize their code, reverse-engineer compiled code, and offers control over
program execution not available using higher level languages.


x86 Architecture

The x86 architecture powers the majority of personal desktop and laptop computers today.
Most mobile devices such as smartphones and tablets are powered by the ARM architecture.
