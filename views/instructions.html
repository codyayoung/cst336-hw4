Reading/Writing Memory

Some of the most basic instructions in x86 assembly are reading and writing memory.
The mov instruction writes data from one register into another.
Moving data between registers is possible, but moving data between memory addresses directly is not.
The push and pop instructions push and pop data from the current stack in memory.
Memory stacks depend on the type of hardware being used.


Arithmetic

Basic math instructions in x86 use two 32-bit registers.
These instructions include addition, subtraction, multiplication, division, and bit shifting.
For example, the following instruction would add the value in the EAX register to the ECX register.

addl %eax, %ecx
